Página inicial — Guia Rápido de Git

Um modelo de página inicial simples e responsiva para apresentar seu guia de comandos do Git. Inclui seções para comandos, exemplos e um botão de download do guia em Word.

Iniciar repositório
git init
Inicia repositório local
Clonar
git clone <url>
Clona repositório remoto
Adicionar
git add .
Adiciona todos os arquivos
Commit
git commit -m "mensagem"
Salva alterações

Resumo rápido

Fluxo recomendado:

  1. git clone → criar branch → fazer alterações
  2. git add → git commit
  3. git push → abrir pull request
